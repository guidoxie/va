## 指数估值表，不构成投资建议，投资有风险，入市需谨慎

### Help
```
➜ va -h
指数估值表，不构成投资建议，投资有风险，入市需谨慎

数据来源:中证指数官网（www.csindex.com.cn）

Usage:
   [flags]

Flags:
  -c, --code string   指数代码
  -d, --date string   日期，格式：2006-12-28
  -f, --fast          快速获取，减少模拟用户行为
  -h, --help          help for this command
  -t, --table         常见指数估值表，日期默认为昨天 (default true)
```

### 估值建议计算方式（仅供参考，不构成投资建议）
#### 价值指数-盈利收益率法
适用于流通性比较好，盈利比较稳定的品种

适合指数：
* 300价值（000919）
* 上证50（000016）
* 上证红利（000015）
* 中证红利（000922）
* 基本面50（000925）
* 上证180（000010）

策略：
* 盈利收益率大于10%（大幅高于同期无风险利率，如：十年期国债），为低估，建议分批定投
* 盈利收益率小于10%，但大于6.4%时，正常估值，建议持有
* 盈利收益率小于6.4%时，高估值，建议分批卖出


#### 成长指数-博格公式法（市盈率）
适用于盈利高速增长的，或者是盈利呈周期性变化的品种

适合指数：
* 沪深300（000300）
* 中证500（000905）
* 养老产业（399812）
* 可选消费（000989）
* 医药100（000978）
* 中证消费（000932）
* 500低波动（930782）
* 科创50（000688）
* 深证F120（399702）
* 深证F60（399701）

策略：
* 市盈率处于历史波动范围的较低区域，一般这种时候股息率高，建议分批定投
* 市净率处于历史波动范围的中等区域，建议持有
* 市净率处于历史波动范围的较高区域，建议分批卖出

#### 周期指数-博格公式的变种（市净率）
适用强周期性品种

* 适合指数：
* 证券公司（399975）
* 中证银行（399986）

策略：
* 市净率处于历史波动范围的较低区域，建议分批定投
* 市净率处于历史波动范围的中等区域，建议持有
* 市净率处于历史波动范围的较高区域，建议分批卖出
